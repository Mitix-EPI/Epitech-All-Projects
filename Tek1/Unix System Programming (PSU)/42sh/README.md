# 42sh

![Badge Mark](https://img.shields.io/badge/Module%20Mark-A-%2372FA08.svg?&style=for-the-badge&logoColor=black)

## Overview

The 42sh is based on the TCSH shell.

42sh is composed of:
 - Environment variables
 - `cd` command
 - Exec environment commands
 - `|` `&` `||` `&&` `>` `>>` `<<` `<` separators
 - globing
 - var interpreter
 - alias
 - repeat
 - terminal shortcuts
